# ⚙️ Getting Started

## Clone Repository

```bash
git clone https://github.com/vaibhavpokhriyal13/Portfolio.git
```

## Navigate to Project

```bash
cd Portfolio
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

Visit:

```text
http://localhost:5173
```

# ☁️ Deployment

The portfolio is deployed on **Vercel**.

### 🌐 Live Website

**https://vaibhav-portfolio-three.vercel.app**

To deploy your own version:

1. Fork this repository.
2. Import it into Vercel.
3. Install dependencies automatically.
4. Configure environment variables (if required).
5. Deploy with a single click.

Every push to the production branch automatically triggers a new deployment.
